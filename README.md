##  

## Requires

MySQL, npm|node.js

## Project setup

Rename .env copy to .env and fill DATABASE_URL and JWT_SECRET

```bash
$ npm install
```
## Documentation

(domain)/docs

## Compile and run the project


```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
